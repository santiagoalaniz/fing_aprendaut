# Ejercicio 3

Espacio de hipotesis `< x, x, x, x, x >  v  < x, x, x, x, x >`

## Parte (a)

Como se trata de la union de dos espacios de hipotesis identicos, el tamaño de H es el doble del tamaño de cada uno de ellos.

$
|H| = 2 * |< x, x, x, x, x >| = 40960
$
